# SummarizeMe

A quick guide on how to use our SummarizeMe telegram AI bot.

## Getting Started

1. Install the required dependencies

   ```bash
   pip install google.generativeai
   pip install telebot
   ```
2. Run the bot
   ```bash
   python main.py
   ```
   
3. Message the bot on telegram

   ```bash
   @SummarizeMe_bot
   ```

4. Enjoy interacting with SummarizeMe
